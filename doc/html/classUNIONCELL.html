Detailed Description

cell의 기본 형태를 상속하고 MISO의 형태를 갖고 있는 union cell이다. (processing cell이 아니다.) MISO의 형태를 갖는다.

cell의 기본 형태를 상속하고 MISO의 형태를 갖고 있는 union cell이다. (processing cell이 아니다.) MISO의 형태를 갖는다. 단순히 두개 이상의 튜플을 합쳐준다. 예를들어 하나의 튜플은 [int, char] 데이터를 갖고 있고 다른 하나의 튜플은 [char, char]를 갖고 있으면 합쳐지면 [int, char, char, char]의 튜플을 갖게 된다.

Parameters
inpipelistcell에 대한 input pipe list이다. 깊은 복사로 넘어간다. 이후 파라미터로 들어온것은 필요 없으면 메모리 해제 해야 한다.

outpipecell에 대한 단일 output pipe이다.
countcell에 들어있는 worker의 개수를 정의한다. worker는 단순히 두개 이상의 튜플을 합치는 일을 한다.
parentcell이 속한 factory를 명시한다.
