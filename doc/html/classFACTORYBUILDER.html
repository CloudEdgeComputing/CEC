void FACTORYBUILDER::submitScriptforFactory ( )

팩토리를 만들 수 있게 스크립트를 받을 수 있는 형태가 되도록 만든다.

이 함수를 실행하면 스크립트를 입력 받을 수 있는 형태가 된다. 메인 스크립트 당 하나의 팩토리를 확보하고 스크립트를 통해 셀들을 설치하여 완전한 하나의 팩토리를 구성해야 한다.
